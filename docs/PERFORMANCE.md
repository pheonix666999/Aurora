# Performance

No performance measurement has run because a compiler and executable are unavailable. CPU model, RAM, compiler, Release build, and observed processing use therefore remain **NOT MEASURED**.

Required measurement matrix after a verified build: Windows x64 and macOS on both Apple Silicon and Intel where hardware is available; 44.1/48/96 kHz; 512 samples; analyzer enabled and disabled; representative stereo programme material for at least 30 minutes. Record build, CPU model, RAM, compiler/Xcode version, sample rate, buffer, analyzer state, mean/maximum processor CPU, glitches, and denormal/silence behavior. Do not compare Debug numbers with Release.
