# Legacy format options

The stable core intentionally targets VST3 x64 and standalone x64. A separately scoped future project could investigate VST2 only when a proper licence/SDK is legally available, an original WinAmp DSP adapter, a dedicated host bridge, standalone routing through a supported virtual audio device, or a host-specific adapter. No unofficial VST2 SDK is downloaded or bundled, and no legacy-format support is claimed.
